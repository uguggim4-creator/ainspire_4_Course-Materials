지금 당장 쓸 수 있는 이미지 도구
처음엔 웹에서 바로 써보고, 마음에 들면 구독하세요.
Midjourney — 이미지 생성
시네마틱하고 예술적인 스타일의 이미지를 생성. AI 이미지 생성의 필수 툴이었지만, 현재는 GPT Image, Gemini Image 등의 대안으로 예전만큼 절대적이지 않음. Discord 기반으로 시작되었으나, 현재는 웹에서 사용 가능.
웹/Discord 지원 · 유료 구독
Gemini Image (나노바나나) — 이미지 생성
Gemini 앱 안에서 자연어로 이미지 생성·수정. 인물/반려동물의 일관성을 유지한 편집, 배경 변경, 여러 사진 합성처럼 초보자가 말로 지시하기 좋은 작업에 강점.
Gemini 웹/앱 · 무료/유료
GPT Image 2 — 이미지 생성·편집
OpenAI의 최신 이미지 생성 모델. 텍스트와 이미지를 입력으로 받아 고품질 이미지 생성과 편집을 처리하며, ChatGPT에서 대화하듯 수정 지시를 이어가기 좋음.
ChatGPT / OpenAI API · 무료/유료
Topaz — 업스케일·화질 개선
생성한 이미지나 영상을 최종 납품용으로 키우고 선명하게 만드는 후처리 도구. 노이즈, 블러, 저해상도 문제를 줄일 때 사용.
데스크톱 앱 · 유료
근거: Midjourney Web vs Discord, Google Gemini Nano Banana, OpenAI GPT Image 2, Topaz Labs. Midjourney 사용량 감소는 공식 수치가 아니라 검색량/시장 대안 확산 기준의 해석으로 다룹니다.

Video Tools
이미지를 영상으로 — Kling · Seedance
둘 다 I2V(image to video)에 자주 쓰입니다. 처음엔 한 장면을 짧게 만드는 것부터 시작합니다.
Kling — 이미지 → 영상
이미지 한 장에 움직임과 카메라 이동을 붙여 짧은 영상을 만드는 데 사용. 인물, 제품, 풍경 컷을 자연스럽게 움직이는 용도로 입문자가 접근하기 좋음.
Text to Video · Image to Video
Seedance — 영상 생성·확장
ByteDance의 영상 모델. 텍스트와 이미지 기반 생성, 멀티샷, 카메라 움직임, 레퍼런스 기반 장면 구성에 강점. 여러 컷을 이어가는 영상 실험에 적합.
Text/Image to Video · 멀티샷
초보 워크플로
1. Gemini/GPT Image로 기준 이미지 생성 → 2. Kling 또는 Seedance에 업로드 → 3. 움직임 프롬프트 입력 → 4. 5초 클립으로 테스트.
한 번에 긴 영상 만들지 않기
움직임 프롬프트 예시
camera slowly pushes in, hair gently moves in the wind, the product rotates slowly, soft light reflection, cinematic motion, stable face.
동작 + 카메라 + 안정성
근거: Kuaishou Kling 공식 발표, ByteDance Seedance 모델 문서, Seedance 2.0 공식 블로그.
Sound Tools
영상의 분위기를 완성하는 사운드 도구
BGM은 Suno, 내레이션과 효과음은 ElevenLabs로 나눠 생각하면 쉽습니다.
Suno — 배경음악·노래 생성
장르, 분위기, 가사 아이디어를 텍스트로 입력해 음악을 생성. 짧은 광고 영상, 릴스, 쇼츠의 BGM 후보를 빠르게 뽑을 때 사용.
Text to Music
ElevenLabs — 음성·효과음 생성
텍스트를 자연스러운 음성으로 바꾸거나, 프롬프트로 효과음을 생성. 제품 소개 내레이션, 캐릭터 대사, 장면 전환 효과음에 적합.
Text to Speech · Sound Effects
초보 조합
Suno로 15~30초 BGM 후보 생성 → ElevenLabs로 1문장 내레이션 생성 → 편집툴에서 영상 길이에 맞춰 볼륨 조절.
BGM 먼저, 목소리는 나중
주의할 점
특정 가수·성우·브랜드 사운드를 그대로 따라 하라고 요청하지 않기. 공개/상업 사용 전에는 각 서비스의 라이선스 조건 확인.
권리 확인 필수
근거: Suno 공식 서비스, ElevenLabs 소개, ElevenLabs Sound Effects.
Common Mistakes
왕초보가 자주 하는 실수 TOP 5
1
한 번에 완성작 나오길 기대함 — 프롬프트는 3~10번 수정이 기본입니다. → 반복을 과정으로 받아들이기
2
프롬프트를 너무 길게 씀 — 단어가 많다고 좋아지지 않아요. → 핵심 키워드 5~8개로 압축
3
레퍼런스 없이 바로 생성 — 목표 이미지 없으면 어떤 결과도 만족스럽지 않음. → 참고 영상 먼저 10개 수집
4
도구만 여러 개 가입하고 아무것도 안 함 — 하나를 깊게 써야 실력이 늘어요. → 이미지 도구 하나로 한 달 파기
5
첫 결과물이 마음에 안 들어서 포기 — 영상 한 편이 아니라 장면 하나부터. → 컷 1개 완성을 목표로
Reference
레퍼런스 수집이 실력의 80%
Pinterest
분위기·색감·구도 레퍼런스. 영어로 검색하면 훨씬 많이 나옴. 보드에 저장해서 카테고리 관리.
추천 검색어: cinematic portrait, luxury product shot
Instagram Reels / TikTok
최신 트렌드 영상 레퍼런스. 마음에 드는 영상 저장 → 어떤 움직임인지 분석.
AI 영상 계정 팔로우 권장
Civitai / PromptHero
다른 사람들의 AI 이미지 + 프롬프트 공개. 맘에 드는 이미지 클릭하면 프롬프트 바로 확인 가능.
프롬프트 학습에 최고
나만의 레퍼런스 폴더
분위기별 폴더 3개 이상 만들기. ex) 인물·제품·풍경. 저장한 이미지에서 공통 키워드 뽑기.
매주 10장 이상 수집 목표
레퍼런스를 많이 모을수록 "이런 느낌" 을 설명하는 단어가 생깁니다. 그게 곧 프롬프트 실력입니다.

Core Features
Magnific의 진짜 강점은 이 3가지
생성보다는 "이미 가진 이미지를 더 좋게 만드는" 후반 도구로 잡고 가는 게 정확합니다.
Upscale to 10K
2x · 4x · 8x · 16x 업스케일. 최대 10,752 × 7,168 픽셀까지. Creativity 슬라이더로 디테일을 새로 만들어내는 양을 조절합니다.
화질 끌어올리기
Relight
기존 이미지의 조명·배경을 AI로 다시 깔아주는 기능. 같은 인물 사진의 무드를 통째로 바꿀 때 강력합니다.
조명·배경 교체
40+ AI Models · Spaces
한 계정에서 여러 이미지·영상·오디오 모델 사용 + 협업 Spaces. 옛 Freepik 사용자라면 자연스럽게 연속됩니다.
올인원·협업
3개의 핵심 슬라이더
Creativity(AI가 디테일을 새로 만들 정도) · Prompt Strength(텍스트 영향력) · HDR(명암 다이내믹). 이 3개만 알아도 80% 활용.
