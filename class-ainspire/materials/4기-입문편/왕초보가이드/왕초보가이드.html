Tools

Midjourney — 이미지 생성
가장 강력한 이미지 품질. 감성적·영화적 분위기 표현에 탁월. Discord에서 실행. 프롬프트 → 4장 중 선택 방식.
Basic $10/월 · 25장/일
⚡
Flux (Freepik) — 이미지 생성
무료로 시작 가능. 사실적 인물 표현에 강점. 웹 브라우저에서 바로 사용. 처음 시작하기 가장 쉬운 선택.
무료 플랜 있음
🎬
Kling — 이미지 → 영상
이미지를 입력하면 5~10초 영상으로 만들어 줌. 사실적인 움직임이 강점. 쿠앙수아이 개발사의 영상 AI.
무료 66크레딧/일
🎵
Suno — 배경음악 생성
장르·분위기를 텍스트로 입력하면 실제 음악처럼 만들어 줌. 완성된 영상에 BGM을 입힐 때 사용.
무료 50크레딧/일

Common Mistakes
왕초보가 자주 하는 실수 TOP 5
1
한 번에 완성작 나오길 기대함 — 프롬프트는 3~10번 수정이 기본입니다. → 반복을 과정으로 받아들이기
2
프롬프트를 너무 길게 씀 — 단어가 많다고 좋아지지 않아요. → 핵심 키워드 5~8개로 압축
3
레퍼런스 없이 바로 생성 — 목표 이미지 없으면 어떤 결과도 만족스럽지 않음. → 참고 영상 먼저 10개 수집
4
도구만 여러 개 가입하고 아무것도 안 함 — 하나를 깊게 써야 실력이 늘어요. → Flux 하나로 한 달 파기
5
첫 결과물이 마음에 안 들어서 포기 — 영상 한 편이 아니라 장면 하나부터. → 컷 1개 완성을 목표로
