자연어 타이핑만으로 프리미어 프로의 컷 편집·자막·이펙트를 직접 제어하는 MCP 연동 패키지입니다.

🤖 그러면 클로드가 하는 일
1
Node.js 설치 여부 확인 (없으면 설치 안내)
2
설치 스크립트로 의존성 설치 · 빌드 · MCP 등록까지 자동 진행
3
프리미어에서 MCP Bridge 패널 켜는 법 안내
4
연결이 안 되면 진단(doctor)까지 실행해 원인 안내
🎬 설치 후 프리미어에서 창 → 확장 → MCP Bridge → Start Bridge만 켜면, "3초 지점 클립 잘라내고 뒤를 붙여줘"처럼 자연어로 편집을 시킬 수 있습니다.
