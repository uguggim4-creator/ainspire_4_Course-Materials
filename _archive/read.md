# ainspire 4기 강의 자료실

## 목적
이 폴더는 ainspire 4기 강의 자료를 주차별로 정리하기 위한 작업 공간입니다. 강의안, PPT, 캡쳐 이미지, 실습자료, 최종 산출물을 분리해서 관리합니다.

## 폴더 구조

```text
ainspire 4기 강의/
├─ 00_운영_가이드/
│  ├─ 템플릿/
│  ├─ 체크리스트/
│  └─ 참고자료/
├─ 01_0주차_설치_실행_자동화/
│  ├─ 강의안/
│  ├─ PPT/
│  ├─ 이미지_캡쳐/
│  ├─ 실습자료/
│  └─ 산출물/
├─ 02_1주차/
├─ 03_2주차/
├─ 04_3주차/
├─ 05_4주차/
└─ 99_아카이브/
```

## 폴더 사용 규칙

### 1. 주차별 폴더
- 주차별 핵심 자료는 `01_0주차_...`, `02_1주차`처럼 번호를 붙여 정렬합니다.
- 주차명이 확정되면 `02_1주차_주제명`처럼 뒤에 주제를 붙입니다.

### 2. 파일명 규칙
파일명은 아래 형식을 기본으로 사용합니다.

```text
[주차]_[자료종류]_[내용]_[버전].확장자
```

예시:

```text
0주차_강의안_설치실행자동화_v0.1.md
0주차_PPT_설치실행자동화_v0.1.pptx
0주차_체크리스트_설치준비_v0.1.md
```

### 3. 버전 규칙
- 초안: `v0.1`, `v0.2`
- 강의 전 확정본: `v1.0`
- 강의 후 수정본: `v1.1`
- 오래된 파일은 삭제하지 않고 `99_아카이브`로 이동합니다.

### 4. 이미지 캡쳐 규칙
캡쳐 이미지는 주차별 `이미지_캡쳐` 폴더에 저장합니다.

```text
01_codex_install.png
02_codex_first_run.png
03_claude_app_main.png
04_vscode_extension.png
05_antigravity_main.png
```

주의사항:
- 이메일, API 키, 결제 정보, 개인 폴더명은 가리고 캡쳐합니다.
- PPT용 이미지는 가능하면 16:9 비율, 1920x1080 이상으로 준비합니다.
- 한 장에 너무 많은 정보를 넣지 않고, 메뉴와 버튼이 보이도록 여백을 둡니다.

### 5. 최종 산출물 규칙
- 강의 중 수정 가능한 파일은 원본 형식으로 보관합니다. 예: `.md`, `.pptx`, `.docx`, `.html`
- 배포용 파일은 `산출물` 폴더에 저장합니다. 예: `.pdf`, 최종 `.pptx`

## 0주차 핵심 산출물
- 0주차 강의내용 정리.md
- 설치 체크리스트
- 도구별 UI 캡쳐 이미지
- 강의용 PPT
- 실습용 예제 파일

## 작업 흐름
1. `강의안` 폴더에서 Markdown으로 내용을 먼저 정리합니다.
2. 캡쳐가 필요한 화면은 `이미지_캡쳐` 폴더에 넣습니다.
3. 강의안과 이미지를 기반으로 `PPT` 폴더에 발표자료를 만듭니다.
4. 수강생에게 배포할 파일은 `산출물` 폴더에 따로 저장합니다.
