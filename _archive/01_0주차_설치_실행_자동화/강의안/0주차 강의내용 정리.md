# 0주차 강의내용 정리: 설치 / 실행 / AI 작업 환경 가이드

## 문서 목적

이 문서는 ainspire 4기 0주차 강의를 실제 수업 흐름에 맞춰 진행하기 위한 강의 목차입니다.
세부 설명문, 예시 문장, 캡쳐 이미지는 이후 각 목차별 세부 문서와 PPT 제작 단계에서 채웁니다.

> 운영 원칙: 이 목차에서는 자동화 파트를 다루지 않는다. 이후 세부 내용과 PPT 구성에서도 자동화 섹션을 추가하지 않는다.
> 핵심 원칙: 개념 간 층위 구분을 명확히 한다. 실행 환경, 실행 방식, 연결 방식, 작업 절차, 기능 확장, 산출물 형식을 섞어 설명하지 않는다.
