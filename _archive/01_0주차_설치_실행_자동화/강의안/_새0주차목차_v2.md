## 5. 네 가지 도구 설치·첫 실행: 한 섹션에서 빠르게 끝내기

(목표: Codex, Claude 앱, VS Code, Antigravity를 모두 설치·첫 실행하고, 각 도구의 역할 차이를 구분할 수 있다.)

### 5.1 진행 원칙: 네 도구 모두 설치·첫 실행, 막히면 표시하고 넘어가기  
설치가 수업 전체를 잡아먹지 않도록 이 섹션은 타임박스로 진행한다. 실패한 도구는 9장의 백업 플랜으로 처리한다.

### 5.2 Claude 앱 실행: 대화형 AI 작업 앱  
새 대화 화면까지 확인한다. 제공 방식, 설치 여부, 로그인 절차, 모델명은 **강사가 공식문서로 확인 필요**.

### 5.3 VS Code 실행: 파일을 직접 보고 편집하는 편집기, IDE  
강의 폴더를 열고 새 파일을 저장할 수 있는지 확인한다. 설치 화면과 OS별 권한 문구는 **강사가 공식문서로 확인 필요**.

### 5.4 Codex 실행: AI가 파일 작업을 돕는 작업 환경  
첫 실행, 로그인, 작업 화면 확인까지만 진행한다. 0주차에서는 자동 수정, 대량 변경, 명령 실행을 하지 않는다. 정확한 기능·제공방식은 **강사가 공식문서로 확인 필요**.

### 5.5 Antigravity 실행: 프로젝트형 AI 작업 환경  
첫 실행, 프로젝트/작업 화면 확인까지만 진행한다. 0주차에서는 자동화나 외부 서비스 연결을 하지 않는다. 정확한 기능·제공방식은 **강사가 공식문서로 확인 필요**.

### 5.6 브라우저 준비: HTML 파일 확인 용도로만 사용

### 5.7 오늘의 기본 설정: 모델 비교 없이 기본 설정으로 진행  
모델명, 요금, 사용량 제한, 가격표는 오늘 분석하지 않는다. 필요한 정보는 **강사가 공식문서로 확인 필요**.
