05

대화와 결과물은 다르다

왜 작업형 AI가 필요한가
대화창 = 임시

대화형 사용

“블로그 글 써줘” → 화면에 텍스트만 출력. 화면을 닫으면 사라지고, 복사해서 직접 옮겨야 한다.

복사 필요휘발됨
→
파일 = 영구

작업형 사용

“blog-draft.md로 저장해줘” → 내 작업 폴더에 파일 생성. 언제든 다시 열어 수정할 수 있다.

파일 생성저장·재사용
